Definition: Game

A game is defined by:

  1. Set of players: \(i \in \{1, 2, \ldots, n\} = I\)
  2. Set of strategies \(s_i \in S_i\) for each player \(i\)
  3. Payoffs for each player \(i\) for each strategy profile \(s = (s_1, s_2, \ldots, s_n) = (s_i, s_{-i})\): \(u(s_i, s_{-i})\)

Here \(s_i\) is player \(i\)’s strategy and \(s_{-i} = (s_1, s_2, \ldots, s_{i-1}, s_{i+1}, \ldots, s_n)\) denotes the vector of other players’ strategies.

Dominance

Dominance

Definition: Strict Dominance

Strategy \(s_i\) strictly dominates \(s_i'\) if for EVERY strategy profile of the others \(s_{-i}\), player \(i\) strictly prefers \(s_i\) over \(s_i'\):

\[u_i(s_i; s_{-i}) > u_i(s_i'; s_{-i})\]

We also say that \(s_i'\) is strictly dominated by \(s_i\). If \(s_i\) strictly dominates ALL other strategies of player \(i\), then we say that \(s_i\) is the strictly dominant strategy for player \(i\).

Definition: Weak Dominance

Strategy \(s_i\) weakly dominates \(s_i'\) if for EVERY strategy profile of the others \(s_{-i}\), player \(i\) prefers \(s_i\) over \(s_i'\):

\[u_i(s_i; s_{-i}) \geq u_i(s_i'; s_{-i})\]

We also say that \(s_i'\) is weakly dominated by \(s_i\). If \(s_i\) weakly dominates ALL other strategies of player \(i\), then we say that \(s_i\) is the weakly dominant strategy for player \(i\).

Best Response

Best Response

Definition: Best Response

For payoff function \(u_i(s_i, s_{-i})\), strategy \(s_i\) is a best response to strategy \(s_{-i}\) if it maximizes player \(i\)’s utility:

\[u(s_i, s_{-i}) > u(s_i', s_{-i})\]

for all strategies \(s_i'\). The set of all best responses is denoted by \(BR_i(s_{-i})\).

Theorem

If strategy \(s_i'\) is strictly dominated (by strategy \(s_i\)), then \(s_i'\) is not a best response to any strategy \(s_{-i}\).

Is the opposite statement true? I.e., if a strategy is never a best response, then it is dominated?

Definition: Dominance Solvable

A game is called dominance solvable if each player has exactly one strategy \(s_i\) left after iterated elimination of strictly dominated strategies.

Nash Equilibrium

Nash Equilibrium

Definition: Nash Equilibrium

A Nash equilibrium is a collection of strategies for each player, \((s_1^*, s_2^*, \ldots, s_n^*)\), such that for each player \(i\):

\[s_i^* \in BR_i(s_{-i}^*)\]

Theorem 2

The solution of a dominance solvable game is a Nash equilibrium of the game.

Mixed Strategies

Definition: Mixed Strategy

A mixed strategy \(\sigma_i\) is a probability distribution over \(S_i\). The probability with which strategy \(s_i\) is chosen in the mixed strategy \(\sigma_i\) is denoted by \(\sigma_i(s_i)\). An expanded strategy set \(\Delta S_i\) consists of all possible randomizations over \(S_i\).

Definition: Best Response to Mixed Strategies

Mixed strategy \(\sigma_i\) is a best response to mixed strategies \(\sigma_{-i}\) if all pure strategies \(s_i\) used by \(\sigma_i\) (that is, all \(s_i\) such that \(\sigma_i(s_i) > 0\)) are best responses to \(\sigma_{-i}\):

\[s_i \in BR_i(\sigma_{-i})\]

or equivalently \(u(s_i, \sigma_{-i}) > u(s_i', \sigma_{-i})\) for all other \(s_i'\).

Mixed Strategy Nash Equilibrium

Theorem

Strategy \(s_i\) is not a best response to any strategy \(\sigma_{-i}\) of other players if and only if it is strictly dominated, possibly by a mixed strategy.

Definition: Mixed Strategy Nash Equilibrium

A mixed strategy Nash equilibrium is a collection of mixed strategies \((\sigma_1^*, \ldots, \sigma_n^*)\) such that:

\[\sigma_i^* \in BR_i(\sigma_{-i}^*) \quad \text{for each } i\]

Summary

Week 11: Key Takeaways

  1. A game is defined by players, strategy sets, and payoff functions

  2. A strictly dominant strategy is the best response to every possible opponent strategy

  3. Iterated elimination of strictly dominated strategies can sometimes solve a game

  4. A Nash equilibrium is a strategy profile where every player is playing a best response

  5. Some games have no pure strategy NE, motivating mixed strategies – probability distributions over pure strategies

  6. A mixed strategy NE always exists (Nash’s Theorem)
