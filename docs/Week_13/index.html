Asymmetric Information

Math Recap: Bayes’ Rule

Suppose that \(A\) and \(B\) are stochastic events, then if \(P(B)>0\):

\[P(A|B)=\frac{P(A\cap B)}{P(B)}=\frac{P(B|A)P(A)}{P(B)}\]

Suppose that \(\Omega\) is the whole outcome space. Events \(A_{1},A_{2},\dots,A_{n}\) are called a partition if for any \(i\neq j\): \(A_{i}\cap A_{j}=\emptyset\) and \(\cup_{i=1}^{n}A_{i}=\Omega\). If \(P(B)>0\), then

\[P(A_{j}|B)=\frac{P(B|A_{j})P(A_{j})}{P(B)}=\frac{P(B|A_{j})P(A_{j})}{\sum_{i=1}^{n}P(B|A_{i})P(A_{i})}\]

Math Recap: Conditional Expectation (Discrete)

Suppose that a random variable \(X\) takes values \(x_{1},x_{2},\dots,x_{n}\) with probabilities \(P(X=x_{i})=p_{i}\) for any \(i=1,2,\dots,n\). Then, its expectation is

\[\mathbb{E}[X]=\sum_{i=1}^{n}x_{i}p_{i}\]

Now consider some condition \(A\) (e.g., \(X<10\)), then the conditional expectation given \(A\) is

\[\mathbb{E}[X|A]=\sum_{i=1}^{n}x_{i}\,P(X=x_{i}|A)\]

Math Recap: Conditional Expectation (Continuous)

A continuous random variable \(X\) has pdf \(f(x)\) and cdf \(F(x)=P(X\leq x)=\int_{-\infty}^{x}f(t)\,dt\). Its expectation is \(\mathbb{E}[X]=\int x\,f(x)\,dx\).

The conditional expectation given an event \(A\) is

\[\mathbb{E}[X|A]=\int x\,f_{X|A}(x)\,dx=\frac{\int_{A}x\,f(x)\,dx}{\int_{A}f(x)\,dx}\]

Note that \(f_{X|A}(x)=\frac{f(x)}{P(X\in A)}\) if \(x\in A\). Be careful with the support of \(X\), because it gets restricted by \(A\).

Uniform Distribution

If \(X \mid a\leq X\leq b\) is distributed uniformly on \([c,d]\), then \(\mathbb{E}[X \mid a\leq X\leq b]=\frac{c+d}{2}\).

Adverse Selection

Definition: Adverse Selection

Adverse selection is a phenomenon when a better-informed party selectively participates in trade by often offering only a low-quality product.

Similar partial or complete market failure arises in:

  • Investments into start-ups
  • Insurance markets
  • Labor markets

Takeover Game

  • The buyer can improve the value of an existing company by 50%
  • \(v\) is the exact value of the company, known only to the seller (seller’s type)
  • If the seller of type \(v\) accepts an offer of price \(p\) to sell the company, then the buyer receives \(1.5v - p\), whereas the seller gets \(p\)
  • If the seller rejects the offer, the buyer receives \(0\) and the seller retains the company at its current value \(v\)

The buyer knows the value of the company. Will the trade happen?

If the buyer does not know the value, will there be trade?

Labor Market

  • Each worker has productivity \(\theta\in[0,1]\), which is private information
  • All workers have an outside option (reservation wage) \(r(\theta)=0.75\theta\)
  • The labor market is competitive, so the firms receive zero profit

Productivity is observed by both parties:

  • Firm’s profit: \(\pi=\theta-w\)
  • Competition \(\Rightarrow\) \(w=\theta\)
  • The worker accepts: \(w=\theta\geq r(\theta)=0.75\theta\)
  • Thus, it is efficient for any type of the worker to be hired

What happens if the productivity is not observed by the firm?

Moral Hazard

Moral Hazard: Hidden Action

Principal–Agent problem:

  • A principal is an owner of a firm
  • An agent is a manager
  • The profit of the firm depends on effort of the manager \(e\) and a random component (economic conditions) \(\varepsilon\): \(\pi = e + \varepsilon\)
    • The owner observes the profit, but does not know effort \(e\)
    • The only information about \(\varepsilon\) for the owner: \(\mathbb{E}[\varepsilon]=0\)
  • The owner maximizes: \(u_{p}(\pi, s) = \mathbb{E}[\pi - w(\pi)]\), where \(w(\pi)\) is wage paid to the manager
  • The manager maximizes: \(u_{a}(w, e) = \mathbb{E}[w(\pi)] - c(e)\), where \(c(e)\) is the cost of putting effort \(e\)
    • Suppose \(c(e) = e^{2}\)

Key Question

How should the owner choose the wage scheme \(w(\pi)\) in order to maximize their own utility?
