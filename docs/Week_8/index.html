Part 1: Technology

Production Functions: Definition

The firm uses an input vector \(\mathbf{x} = (x_1, x_2, \dots, x_k) \in \mathbb{R}_+^k\) to produce output \(q \in \mathbb{R}_+\).

Definition: Production Function

A production function \(q = f(\mathbf{x})\) maps input bundles to a scalar output \(q\) — the maximum output the firm can produce from any given combination of inputs.

The partial derivative \(\frac{\partial f(\mathbf{x})}{\partial x_i}\) is the rate of change of output with respect to input \(x_i\), holding all other inputs fixed.

Marginal Products

Definition: Marginal Product

The marginal product of input \(i\) is:

\[MP_i = \frac{\partial f(\mathbf{x})}{\partial x_i}\]

Interpretation: additional output from one more unit of input \(i\), holding others fixed.

Diminishing marginal product: \(\frac{\partial^2 f(\mathbf{x})}{\partial x_i^2} < 0\) — each extra unit of an input contributes less and less to output.

Isoquants

An isoquant is the set of all input vectors that provide the same level of output — the firm’s analogue of indifference curves. Slope \(= -MRTS_{ij}\).

Marginal Rate of Technical Substitution

Definition: MRTS

The marginal rate of technical substitution (MRTS) measures how many units of input \(x_j\) the firm is willing to exchange for one additional unit of input \(x_i\):

\[MRTS_{ij} = -\left.\frac{dx_j}{dx_i}\right|_{q = \text{const}}\]

Theorem: MRTS as Ratio of Marginal Products

\[MRTS_{ij} = \frac{MP_i}{MP_j} = \frac{\frac{\partial f(\mathbf{x})}{\partial x_i}}{\frac{\partial f(\mathbf{x})}{\partial x_j}}\]

(The proof is left as a homework exercise.)

MRTS Along an Isoquant

Drag the point to see how the MRTS changes — it decreases as \(L\) increases (diminishing MRTS).

Properties of a “Typical” Production Function

A well-behaved production function typically satisfies:

  1. Monotonicity: \(MP_i > 0\) for all \(i\)
  2. Diminishing marginal productivity: \(\frac{\partial^2 f(\mathbf{x})}{\partial x_i^2} < 0\)
  3. Convexity of technology (quasi-concavity of \(f\)): \[f(\mathbf{x_1}) = f(\mathbf{x_2}) = q \;\Rightarrow\; f(\alpha \mathbf{x_1} + (1-\alpha)\mathbf{x_2}) \geq q\]
  4. Diminishing MRTS: Moving along an isoquant, the MRTS decreases

Cobb-Douglas Production Function

\(f(L,K) = A L^a K^b\). \(\;MP_L = aAL^{a-1}K^b\), \(\;MP_K = bAL^{a}K^{b-1}\), \(\;MRTS = \frac{a}{b}\frac{K}{L}\).

CES Production Function

\(f(L,K) = (\frac{1}{2}L^\rho + \frac{1}{2}K^\rho)^{1/\rho}\), where \(\rho\) controls the elasticity of substitution between inputs.

  • High \(\rho\): inputs are easily substitutable (isoquants nearly linear)
  • \(\rho \to 0\): Cobb-Douglas case
  • Low \(\rho\): inputs are hard to substitute (isoquants nearly L-shaped)

Returns to Scale

Definition: Returns to Scale

A technology exhibits:

  • DRS: \(f(t\mathbf{x}) < t \cdot f(\mathbf{x})\) for \(t > 1\) — diseconomies of scale
  • CRS: \(f(t\mathbf{x}) = t \cdot f(\mathbf{x})\) for \(t > 1\) — scale-neutral
  • IRS: \(f(t\mathbf{x}) > t \cdot f(\mathbf{x})\) for \(t > 1\) — economies of scale

Real-world: IRS (gas pipes, social networks), CRS (gyms), DRS (management inefficiencies)

Returns to Scale: Examples

  1. \(f(\mathbf{x}) = x_1^2 + x_2^2\): \(\;f(t\mathbf{x}) = t^2 f(\mathbf{x})\) → IRS
  2. \(f(\mathbf{x}) = \sqrt{x_1^2 + x_2^2}\): \(\;f(t\mathbf{x}) = t \cdot f(\mathbf{x})\) → CRS
  3. \(f(\mathbf{x}) = \sqrt{x_1 + x_2}\): \(\;f(t\mathbf{x}) = t^{0.5} f(\mathbf{x})\) → DRS

Part 2: The Firm’s Problem

The Firm’s Objective: Profit Maximisation

Key Assumption

Firms maximise profit: \(\pi = \text{Revenue} - \text{Cost}\)

With output price \(p\) and input price vector \(\mathbf{w} = (w_1, \dots, w_n)\):

\[\max_{\mathbf{x}} \; p \cdot f(\mathbf{x}) - \mathbf{w} \cdot \mathbf{x}\]

The solution depends on the market structure.

Two-Step Decomposition

  1. Cost Minimisation (Production Department): \[\min_{L,K} \; wL + rK \quad \text{s.t.} \; f(L,K) = q\] → Conditional demands \(L^c(q), K^c(q)\); cost function \(c(q) = wL^c + rK^c\)

  2. Profit Maximisation (Marketing Department): \[\max_q \; p \cdot q - c(q)\] → Optimal output \(q^*\); supply function \(q(p)\)

Market Structures

Definition

A market structure summarises the degree of competitiveness or market power.

  • Perfect competition: \(N\) large, firm is a price-taker, \(p\) is constant
  • Oligopoly: \(N\) small, strategic interaction
  • Monopoly: \(N = 1\), firm sets price

For the rest of this unit, we focus on the competitive (price-taking) firm.

Running Example: \(f(L,K) = L^{1/3}K^{1/3}\)

We will use this production function throughout Parts 3 and 4 as our running example.

  • DRS: \(f(tL, tK) = t^{2/3} \cdot f(L,K)\) (homogeneous of degree \(2/3\))
  • Diminishing \(MP_L\) and \(MP_K\)
  • \(MRTS = K/L\)

Part 3: Cost Minimisation

Isocost Lines

\(c(L,K) = wL + rK\). An isocost line has slope \(-w/r\).

The Cost Minimisation Problem

\(\min_{\mathbf{x}} \mathbf{w} \cdot \mathbf{x}\) s.t. \(f(\mathbf{x}) \geq q\). Find the point on the isoquant on the lowest isocost line. Analogous to the consumer’s EMP.

Tangency Condition and Lagrangian

Lagrangian: \(\mathcal{L} = wL + rK + \lambda[q - f(L,K)]\)

FOCs:

  • \(w = \lambda \cdot MP_L\), \(\quad r = \lambda \cdot MP_K\), \(\quad q = f(L,K)\)

Tangency condition: \(\;\frac{MP_L}{MP_K} = \frac{w}{r}\), i.e., \(MRTS = w/r\).

Interpretation of \(\lambda\): marginal cost of producing one more unit (\(\lambda = w/MP_L = r/MP_K\)).

Cost Minimisation Along an Isoquant

Five input combinations on the \(q = 4\) isoquant for \(f(L,K) = L^{1/3}K^{1/3}\), with different costs.

Running Example: Conditional Factor Demands

For \(f(L,K) = L^{1/3}K^{1/3}\) with input prices \(w, r\):

  1. Tangency: \(MRTS = w/r \;\Rightarrow\; K/L = w/r \;\Rightarrow\; K = (w/r)L\)
  2. Constraint: \(q = L^{1/3}K^{1/3} = (w/r)^{1/3} L^{2/3}\)
  3. Solve: \(L^c = (r/w)^{1/2} \cdot q^{3/2}\), \(\quad K^c = (w/r)^{1/2} \cdot q^{3/2}\)
  4. Cost function: \(c(q) = wL^c + rK^c = 2\sqrt{rw} \cdot q^{3/2}\)

Conditional Demands and the Expansion Path

Conditional demands \(L^c(w,r,q)\), \(K^c(w,r,q)\): cost-minimising inputs for target \(q\). The expansion path traces these as \(q\) varies (analogous to income offer curve).

The Cost Function

\(c(w,r,q) = wL^c + rK^c\) — analogous to the expenditure function. For \(f = L^{1/3}K^{1/3}\): \(c(q) = 2\sqrt{rw} \cdot q^{3/2}\) (convex in \(q\), reflecting DRS).

Part 4: Profit Maximisation

The Marketing Department’s Problem

The marketing department receives \(c(q)\) and solves:

\[\max_q \; p \cdot q - c(q)\]

FOC: \(\;\frac{d}{dq}[pq - c(q)] = 0 \;\Rightarrow\; p = MC\)

Key Result: Competitive Supply

A competitive firm sets price equal to marginal cost: \(p = MC\).

Competitive Firm Profit Maximisation

Supply of Output

From \(p = MC\), derive the firm’s supply function \(q^*(p)\).

The Direct Approach

Instead of two steps, solve profit maximisation directly over inputs:

\[\max_{\mathbf{x}} \; p \cdot f(\mathbf{x}) - \mathbf{w} \cdot \mathbf{x}\]

FOC: \(\;p \cdot MP_i = w_i\) for each \(i\)

\[\underbrace{p \cdot MP_i}_{\text{value of marginal product}} = \underbrace{w_i}_{\text{input price}}\]

SOC: Hessian of \(f(\cdot)\) must be negative semi-definite — i.e., \(f\) must be concave. This holds under DRS (as in our running example), but fails under CRS and IRS.

Solution: \(\mathbf{x}(p, \mathbf{w})\) — factor demand; \(\;q = f(\mathbf{x}(p, \mathbf{w}))\) — supply.

Demand for Labour: \(MRP_L = w\)

From the direct approach, \(p \cdot MP_L = w\) determines the firm’s labour demand \(L^*(w, p)\).

CRS and IRS: Problems for Profit Maximisation

Recall: the SOC requires \(f\) to be concave. What if it isn’t?

If \(\pi > 0\) under CRS or IRS, the firm can scale up all inputs by \(t > 1\):

\[pf(t\mathbf{x}) - \mathbf{w}(t\mathbf{x}) \geq t\pi > \pi \quad \Rightarrow \quad \text{no finite optimum}\]

Example: \(f(x) = x^\alpha\), single input with price \(w\).

Returns to Scale Factor demand \(x^*\) Supply \(q^*\) Profit
IRS (\(\alpha > 1\)) \(\to +\infty\) \(\to +\infty\) \(\to +\infty\)
CRS (\(\alpha = 1\)) any \(x \geq 0\) if \(p = w\) any \(q \geq 0\) \(0\)
DRS (\(\alpha < 1\)) \(\left(\frac{w}{\alpha p}\right)^{\frac{1}{\alpha-1}}\) \(\left(\frac{w}{\alpha p}\right)^{\frac{\alpha}{\alpha-1}}\) \(> 0\), finite

Key Takeaway

For a well-defined finite solution, the technology must exhibit DRS (at least locally).

Summary

Week 8: Key Takeaways

  1. Production technology is described by \(q = f(\mathbf{x})\); MRTS measures the rate of input substitution along isoquants

  2. Special cases — Cobb-Douglas, CES — differ in substitutability and isoquant shape

  3. Returns to scale (DRS, CRS, IRS) describe how output responds to proportional input changes; CRS and IRS create problems for profit maximisation

  4. Cost minimisation (tangency condition: \(MRTS = w/r\)) yields conditional demands, the cost function, and the expansion path

  5. Profit maximisation requires \(p = MC\) (two-step) or \(p \cdot MP_i = w_i\) (direct approach)
